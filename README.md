# DiGCT: Online Thermal Monitoring fo IGCT via Physics-Informed Diffusion Model

## 🧩 Setup Guideline
Please meet the requirement of `environment.yaml`. In general, the following dependencies should be installed
* PyTorch >= 1.6.0
* numpy
* Python >= 3.6
